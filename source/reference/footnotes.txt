=========
Footnotes
=========

Use footnotes to add detail that not all readers will need.

Syntax
------

.. procedure::
    :style: normal

    .. step:: If the page doesn't have a "Footnotes" rubric, add one.

        .. procedure::
            :style: connected

            .. step:: At the bottom of the page, type ``.. rubric:: Footnotes``.

    .. step:: Mark the footnote location in your text. 

        .. procedure::
            :style: connected

            .. step:: Add a space after the end of the sentence you're adding the 
                footnote to.
            
            .. step:: Enter ``[#footnote-identifier]_``, where "footnote-identifer" is 
                whatever you like.

    .. step:: Add the footnote.
        
        .. procedure::
            :style: connected
            
            .. step:: Beneath the "Footnotes" rubric, enter ``.. [#footnote-identifier] 
                <footnote>``

Example
-------

.. code-block:: rst

    This sentence ends with a footnote. [#first-footnote]_
    This sentence ends with a different footnote. [#second-footnote]_
    Two sentences can use the same footnote. [#first-footnote]_

    .. rubric:: Footnotes
    .. [#first-footnote] This footnote supplements the first and third sentences above.
    .. [#second-footnote] This footnote supplements the second sentence above.
